# Feature

Implementa uma nova funcionalidade.

## Instruções

1. Entenda o requisito da feature
2. Planeje a implementação
3. Crie os arquivos/módulos necessários
4. Implemente a feature seguindo os padrões do projeto
5. Adicione testes
6. Rode `cargo test` e `cargo clippy`
7. Documente a nova funcionalidade
